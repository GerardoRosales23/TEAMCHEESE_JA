    Descripción de cada uno de estos herramientasd:

    1. HTML (HyperText Markup Language): HTML es el lenguaje estándar utilizado para crear y diseñar páginas web. Permite estructurar el contenido de una página web mediante etiquetas, que definen diferentes elementos como encabezados, párrafos, imágenes, enlaces, formularios, entre otros. Es el lenguaje base de la web y proporciona la estructura básica para el contenido de una página.
    2. CSS (Cascading Style Sheets): CSS es un lenguaje utilizado para dar estilo y diseño a las páginas web creadas con HTML. Permite controlar aspectos visuales como el color, la tipografía, el espaciado, el diseño de la página y la disposición de los elementos. Con CSS, los desarrolladores pueden separar el contenido de la presentación, lo que facilita la creación de páginas web atractivas y bien organizadas.
    3. Bootstrap: Bootstrap es un framework de desarrollo front-end de código abierto creado por Twitter. Proporciona una colección de herramientas y componentes predefinidos, como botones, formularios, carruseles, barras de navegación, entre otros, que facilitan la creación de interfaces de usuario responsivas y visualmente atractivas. Bootstrap utiliza HTML, CSS y JavaScript para construir interfaces web modernas y adaptables a dispositivos móviles y de escritorio.
    4. JavaScript: JavaScript es un lenguaje de programación interpretado que se utiliza principalmente para agregar interactividad y dinamismo a las páginas web. Permite realizar acciones como validar formularios, manipular el contenido de la página, crear efectos visuales, interactuar con el usuario a través de eventos como clics y desplazamientos, y realizar solicitudes asíncronas al servidor (AJAX). JavaScript es compatible con la mayoría de los navegadores web y es fundamental para el desarrollo de aplicaciones web modernas y dinámicas.
